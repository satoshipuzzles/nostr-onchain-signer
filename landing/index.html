What it does

A Chrome extension that bridges Bitcoin and Nostr in ways nobody else has.

⚡

Social Multi-Sig

Create m-of-n Taproot wallets from your Nostr following list. Your friends' npubs become signing keys — without them even knowing.

✍

Dual Signer

Sign both Bitcoin transactions and Nostr events from one vault. NIP-07 compatible with every Nostr client.

📜

Notes On-Chain

Embed Nostr event IDs into Bitcoin transactions via OP_RETURN. Permanent, censorship-resistant proof on the timechain.

🔐

PSBT Coordination

Send signing requests to co-signers via custom Nostr kinds. Track who has signed and who hasn't in real-time.

👥

Multi-Account

Manage multiple Nostr identities. Switch between accounts like Gmail. Each with its own Taproot address.

🔍

User Discovery

Find active Nostr users across relays. Filter by activity. NIP-50 global search. Build your signing network.

How it works

From npub to multi-sig in four steps.

1

Derive keys from npubs

Every Nostr npub is a valid secp256k1 x-only public key — the same format used by Bitcoin Taproot (BIP341). No conversion needed.

2

Build a Tapscript multi-sig

Select contacts from your following list. Choose a threshold (e.g., 3-of-5). We build a Tapscript using OP_CHECKSIGADD and derive the P2TR address.

3

Coordinate via Nostr

When you want to spend, create a PSBT and send signing requests to co-signers using custom Nostr kinds (9800/9801). They sign and send back.

4

Broadcast to Bitcoin

Once threshold is met, the fully-signed transaction is broadcast. Optionally embed a Nostr note hash in OP_RETURN for on-chain attestation.

Technical stack

Built on proven standards. No custom crypto.

BIP341 Taproot / P2TR addresses
BIP342 Tapscript OP_CHECKSIGADD
BIP340 Schnorr signatures
NIP-07 Browser signer interface
NIP-19 npub/nsec encoding
NIP-50 Relay search protocol
NIP-98 HTTP Auth (uploads)
PSBT Partially Signed Transactions
AES-256 GCM vault encryption
PBKDF2 600k iteration key derivation
OP_RETURN On-chain note embedding
MV3 Chrome Manifest V3 extension

Install

Build from source and load as unpacked extension.

git clone https://github.com/satoshipuzzles/nostr-onchain-signer.git
cd nostr-onchain-signer
npm install
npm run build

Load in Chrome:

  1. Open chrome://extensions
  2. Enable Developer mode (top right)
  3. Click Load unpacked
  4. Select the dist/ folder

Dev mode (hot reload):

npm run dev

Opens standalone at localhost:5173 with mock Chrome APIs for rapid UI development.
